# ForestLegendsGame

![Логотип игры](assetsREADME/logoImg.png)

---
**Игра-история** Примерьте на себя роль школьника, приехавшего на летние каникулы к любимой бабушке. Здесь в самое обычное утро Вы отправитесь в невероятное приключение, а Вашими верными спутниками станут загадки, волшебство и великолепный чёрный кот! Окунитесь в атмосферу леса, где поют птицы и шелестит листва, познакомьтесь лично со знаменитыми сказочными героями вроде Лешего и Бабы Яги. Это игра-история, игра-сказка, где только Вы решаете, каков будет следующий шаг.

**Книга сказок прямо на экране!** 

_Игра рассчитанна на ПК и имеет поддержку книжной и альбомной ориентации у смартфонов._

## Как запустить

- Склонируйте репозиторий. 
- Установите зависимости: npm install
- Запустите сервер: node server.js
- В строке браузера введите: http://localhost:3000



## Описание игры

`После открытия http://localhost:3000,` Вас встречает главная страница с изображением специалиста по дестабилизации лесного спокойствия, призывающего нажать на стартовый жёлудь. Подчинившись указаниям, Вы запустите игру.

`Далее следует` подтвердить своё намерение и _нажать кнопку_ с однозначным именем **"Начать свою историю"**.

**Поздравляю! Игра началась.**
Для перехода к следующему эпизоду следует _кликать по кнопке_ **"Продолжить"**, расположенной в нижнем правом углу экрана. 

**В ходе игры Вы можете:**

-  `читать увлекательный сюжет:` каждый эпизод сопровождается соответствующим изображением происходящего;
-  `наслаждаться звуковым сопровождением:` пение птиц, звуки животных, инструментальная музыка - всё это для лучшего погружения в сказочную атмосферу;
-  `выбрать главного героя:` персонажа, чьей судьбой распоряжаетесь собственноручно;
-  `принимать решения, круто меняющие сюжет:`следует _кликнуть по выбранному варианту_, а затем _на кнопку_ **"Продолжить"**.

**Приятный бонус:** после завершения игры можно с лёгкостью переписать историю и _пройти её заново_, принимая другие решения. Соответствующая _кнопка для клика_ расположенна всё _в_ том же _правом нижнем углу_ экрана.

Скорее напишите свою _лесную легенду_!
